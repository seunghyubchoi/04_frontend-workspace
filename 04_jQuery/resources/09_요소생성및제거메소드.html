* 삽입 관련 메소드, part1

선택된 요소를 기준으로 새로운 요소를 추가시켜주는 메소드

$(A).append(B) : A 요소 내에 뒷 부분에 B를 추가(자손/하위)
$(A).prepend(B) : A 요소 내에 앞 부분에 B를 추가(자손/하위)
$(A).after(B) : A 요소 뒷 부분에 B를 추가(동위)
$(A).before(B) : A 요소 앞 부분에 B를 추가(동위)

* 삽입 관련 메소드, part2

$(B).appendTo(A) : B 요소를 A 요소 내에 뒷 부분에 추가(자손/하위)
$(B).prependTo(A) : B 요소를 A 요소 내에 앞 부분에 추가(자손/하위)
$(B).insertAfter(A) : B를 A요소 뒤에 추가 (동위)
$(B).insertBefore(A) : B를 A요소 앞에 추가 (동위)

* 요소 객체 복제 메소드

변수 = $("선택자").clone([true|false]) : 선택된 요소를 복제해서 반환해주는 메소드
true | false : 복제할 요소에 걸려있는 이벤트까지도 복사할껀지 지정(기본값은 false)
